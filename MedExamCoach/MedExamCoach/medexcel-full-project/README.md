# MedExcel Pro - Medical Training Dashboard

A professional medical training dashboard built with Next.js, TypeScript, and Tailwind CSS. Perfect for medical exam preparation with interactive components and responsive design.

## 🚀 Features

- **Interactive Dashboard** - Complete medical training interface
- **Exam Readiness Gauge** - Visual progress tracking
- **Peer Performance Comparison** - Detailed analytics and benchmarking
- **Study Insights** - Personalized recommendations
- **Responsive Design** - Works on all devices
- **Modern UI** - Clean, professional medical interface

## 🛠️ Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Beautiful icons
- **Vercel** - Deployment platform

## 📦 Installation

1. **Clone or download** this project
2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Open** [http://localhost:3000](http://localhost:3000)

## 🚀 Deployment

### Vercel (Recommended)
1. **Push to GitHub** or **import directly**
2. **Connect to Vercel**
3. **Deploy automatically**

### V0 (Vercel's AI Platform)
1. **Copy the code** from `app/page.tsx`
2. **Paste into V0**
3. **Generate and deploy**

### Static Export
```bash
npm run build
```
The static files will be in the `out` folder.

## 📱 Components

- **Header** - Navigation and user profile
- **Quick Actions** - Practice, Mock Exam, Rapid Review, Flashcards
- **Exam Readiness** - Progress gauge with focus areas
- **Peer Performance** - Comparison tables and analytics
- **Study Insights** - Personalized recommendations

## 🎨 Customization

- **Colors** - Update `tailwind.config.js`
- **Content** - Modify data in `app/page.tsx`
- **Styling** - Edit `app/globals.css`
- **Components** - Add new components in `components/`
